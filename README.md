> sites 为演示文件夹(`public/**/*` 和 `views/**/*` 经过压缩处理);
> static 为前端静态文件夹

## 安装

1. `git clone`
2. `npm install`
3. `bower install`
3. `make`
4. 浏览器打开 `http://localhost:1337/`

> sites 为演示文件夹(public/**/* 和 views/**/* 全部经过 `fis` 压缩处理);
> static 为前端静态文件夹(未压缩)

## 安装

1. `git clone`
2. `cd sites `
3. `npm install`
4. `node app.js`
5. 浏览器打开 `http://localhost:1337/`
